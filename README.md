# battle-of-wizards-2022
Личный проект. Был разработан будучи в школе, будучи абсолютным новичком в программировании. Я не уверен, в каком году он был создан, скорее всего в начале 2022, если не раньше.

Язык: Python

Игра заключается в задачке про трёх волшебников:
На арене находится три волшебника. Цель каждого из них - остаться единственным на арене. Ходы осуществляются по очереди, во время ходов волшебник может предпринять попытку избавиться от одного из волшебников или пропустить ход. Вы ходите первым, но помимо вас есть ещё два волшебника: у первого шанс срабатывания заклинания 70%, у второго - 90%. У вас есть выбор шанса срабатывания заклинания: 60%, 80% или 100%. Если до конца раунда выстоят все волшебники - они проиграют. Волшебники принимают только те варианты, которые выгодны им. 

Все ошибки и неточности в написании кода были учтены в будущем.
